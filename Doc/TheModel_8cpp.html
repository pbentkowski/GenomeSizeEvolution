The main function.

Note that: In the key loop iterating trough time steps environment is being randomly shifted to a new value with a simple do {} while() statement:

do {env += (2*Turb * RNG() - Turb);} while (env * env > 1.0); This statement has a theoretical potential to get stuck and run infinitely. The larger the Turb parameter is, the more probable it gets.

Parameters:
argc- number of parameters at input
argv- the good old char *argv[] (43 parameters)
Returns:
0 if succesfully run
