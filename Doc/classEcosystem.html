Detailed Description

This is 'The Ecosystem'. A place where things are happening.

Stores a vector (STL container) of cell objects. Has methods to set up a population of cells and later feed, kill and reproduce them. Has a lot methods to harvest data.

Core method. Does the horizontal gene transfer.

Called at each time step. Iterates through the population and for each cell in a population tosses if it's gonna be a gene donor. Then picks a cell which gonna be a gene acceptor and in the last step it tosses for each gene in the donor genotype if it will get transfered to the acceptors genotype. Uses cmath floor() for number rounding.

Parameters:
hgt_donor_prob- probability of being a donor
hgt_gene_trans- of one gene to get transfered
