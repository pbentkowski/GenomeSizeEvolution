Introduction

This program is an implementation of a model of the evolution of genome size described in Piotr Bentkowski's Ph.D. thesis:
Modelling evolution of genome size in prokaryotes in response to changes in their abiotic environment (2014) defended at the University of East Anglia.

Compilation

The program has some dependencies on GNU Scientific Library. Should compile smoothly on most GNU/Linux distros with GSL installed. Compilation works fine on Ubuntu 12.04 LTS and 14:04 LTS with a command:

g++ -O2 -o TheModel -L/usr/local/lib TheModel.cpp genotype.cpp gene.cpp cell.cpp ecosystem.cpp rngEngine.cpp tagging_system.cpp -lgsl -lgslcblas -lm

It also worked fine on UEA's GRACE Computational Cluster compiled with Intel C++ Compiler (icc)

Parameters

Program takes exactly 45 parameters. These are:
00 - program's name
01 - the turbulence level (the length of the step in the random walk; between 0 and 1)
02 - resolution of the environment (how dense the environment space is being sampled, well < 1)
03 - the value = 1/[env space surface] (calculate it on a sheet of paper cos it's gonna be faster that way)
04 - a seed for the Mersenne Twister random number generator (a positive integer)
05 - the number of the iterations in the model's run (a positive integer)
06 - the 'not-super-gene' parameter (surface under the Gaussian curve)
07 - the probability of mutation of existing gene (between 0 and 1)
08 - the fraction of env space is occupied by the gene space (suggested value = 1)
09 - the probability of deleting a gene (between 0 and 1)
10 - the probability of duplicating a gene (between 0 and 1)
11 - the maximal number of genes at model's initialisation (don't aim to high, rather dozens then hundreds)
12 - the minimal number of genes at initialisation (must be lower then #11)
13 - the minimal number of genes at all times (at least 1 must be present or it may result in a segmentation fault)
14 - the number of core genes (might be 0)
15 - the cost of maintaining one gene (>= 0)
16 - the gain from maximal fitness (>= 0 , maximal amount of the resource a cell can gain in one go)
17 - the constant metabolic cost (>= 0, a cost of being a cell and being alive)
18 - the minimal resource below which a cell dies
19 - the minimal resource above which a cell divides
20 - the maximal resource bonus given to a cell at the initialisation of the model (actual value is being randomly selected between 0 and this number)
21 - the probability of a random death (between 0 and 1)
22 - the total amount of resource the ecosystem has (a pretty big number, please)
23 - the number of bins in the genome size histogram
24 - the number of bins in the histogram of the envelope of genome size
25 - the width of the bin in the histogram of the envelope of genome size
26 - the number of bins in the histogram of maximums of the fitness function
27 - the width of bin in the histogram maximums of the fitness function
28 - the number of bins in the histogram of the age of cells
29 - the number of bins in the histogram of the resource allocated to cells
30 - the width of bins in the histogram of the resource allocated to cells
31 - the beginning value of the histogram of the resource allocated to cells
32 - the number of bins in the histogram of the resource up-taken by cells in population
33 - the width of bins in the histogram of the resource up-taken by cells in population
34 - the mode of feeding ('s' -sequential, 'p' -proportional; other values will cause the program to prompt an error message and quit)
35 - the parameter how uneven is the resource distribution when a cell is dividing (has meaning only when program works with proportional feeding mode; value between 0 and 0.5)
36 - the number of bins in the histogram of the age at the first reproduction
37 - the width of bins in the histogram of the age at the first reproduction
38 - the number of bins in the histogram of the number of the reproduction events per cell
39 - the width of bins in the histogram of the number of the reproduction events per cell
40 - the number of bins of the histogram of the average intake per unit of time until reproduction
41 - the bin width of the histogram of the average intake per unit of time until reproduction
42 - probability that a cell will become a gene donor in HGT
43 - probability that a gene will get transferred from the donor's genotype to recipient's genotype
44 - time interval (steps) in which statistics are saved to data files

This program can recognise simple errors in the argument list (e.g. value out of range, negative values when positive are needed etc.), but will not recognise when they don't make a 'biological' sense.

Output

Program produces a number of text files containing desired data. The file ModelParams.dat contains the values of parameters fed to the model. The file GenomeSizeData.dat contains the main statistics of the model, including the time stamp used in the visualisation process of the other files with data. This two files are the basics to analyse simulation output. Other files are rather optional and they can be switched off by commenting out some lines in the main function (TheModel.cpp file) what should speed up the program runtime.

Important files:
ModelParams.dat - contains run's parametrisation
GeneralData.dat - contains basic statistics of the output

Each line of these files represents one time step (in sync with GeneralData.dat):
GenomeSizeData.dat - number of genes in genomes; histograms
GeneNumberOfRepr.dat - number of genes in genomes but only of cells which reproduced; histograms
CellsAgeData.dat - age of the cells in population; histograms
AgeTillReproduction.dat - time passed since the last time cell reproduced (or was born); histograms
FrameSizeData.dat - size of the genotype's frame; histograms
FrameMaxData.dat - maximums of the genotype's frame; histograms
AvarGainAtRepr.dat - cells' average total gain before reproduction; histograms
AvarIntakeAtRepr.dat - cells' average intake before reproduction; histograms
RecourceInCells.dat - resources allocated into cells in one time step; histograms
NumberOfReproductions.dat - how many times a cell reproduced; histograms
RecourceUptakenCells.dat - resources taken by cells in one time step; histograms
AvaregeGenotype.dat - U(x) values (intake values) in all the cells in the population; histograms
AvarIntakeAtRepr.dat - cells' average intake before reproduction; histograms
FrameMaxData.dat - maximums of the genotypes' frame; histograms
GeneNumberOfRepr.dat - cells' gene number distribution, only cells which reproduced; histograms
GenotypeEnvelMean.dat - mean values of U(x) function (the genome's envelope)
GenotypeEnvelSTD.dat - SD of U(x) function (the genome's envelope)

Other data files:
MutationRecords.dat - cumulative number of mutations accumulated by each living cell in a defined time period. There is a dedicated function Ecosystem::clearMutationCounters(int current_iteration) for zeroing number of mutation
GenomesOfPopulation.dat - these are all the genomes in the ecosystem

Visualisation

Visualisation is done using Python scripts containing calls to Pylab library (Linux distros will have it in its repos, for Windows you might check the Python(x,y) project). These scripts can be found in 01_PyScripts directory.
